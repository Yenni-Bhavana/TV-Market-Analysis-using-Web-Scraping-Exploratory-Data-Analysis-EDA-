# TV-Market-Analysis-using-Web-Scraping-Exploratory-Data-Analysis-EDA-

## 📌 Project Overview

This project involves collecting real-world TV product data from Flipkart using web scraping techniques and performing Exploratory Data Analysis (EDA) to uncover insights about pricing, ratings, and product trends in the TV market.


## 🎯 Objective

* Collect TV product data from Flipkart
* Clean and preprocess raw scraped data
* Analyze pricing trends and customer ratings
* Understand screen size distribution and product availability


## Data Collection (Web Scraping)

* Extracted data from Flipkart website
* Scraped product details such as:
  * Price
  * Ratings
  * Screen Size
  * Product Specifications
* Converted raw HTML data into structured format using Python


## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* Web Scraping (BeautifulSoup / Requests)


## Dataset Description

The dataset was created through web scraping and includes:

* Product Price
* Customer Ratings
* Screen Size (cm)
* Launch Year
* Additional product attributes


## Steps Performed

### 1. Web Scraping

* Sent requests to Flipkart pages
* Parsed HTML content
* Extracted relevant product information

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted data types
* Cleaned price and rating formats

### 3. Exploratory Data Analysis (EDA)

* Univariate Analysis (distribution of price, ratings)
* Bivariate Analysis (price vs rating, size vs price)
* Outlier detection

### 4. Data Visualization

* Histograms
* Boxplots
* Bar charts
* Scatter plots


## Key Insights

* Most TVs are priced between ₹10,000 and ₹40,000, showing a strong mid-range market.
* Premium TVs exist as outliers with significantly higher prices.
* Customer ratings are mostly above 4.0, indicating overall satisfaction.
* Medium screen sizes are the most commonly available and preferred.
* Recently launched TVs dominate the dataset, reflecting current market trends.


## 📈 Conclusion

This project demonstrates how raw data collected from a real-world source can be transformed into meaningful insights. The analysis highlights key patterns in pricing, customer preferences, and product distribution in the TV market.
